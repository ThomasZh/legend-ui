Table Insert Helper

The table.min.js plugin also offers a tool that allows you to insert rows and columns easier, just by going to the endge of the table.






There are 2 options that are related to the table insert helper works:

  • tableInsertHelper - This option can be used to disable the table insert helper.
  • tableInsertHelperOffset - The distance in pixels from the table border at which to show the insert helper.
